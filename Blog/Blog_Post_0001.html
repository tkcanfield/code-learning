How It's Going

08/01/2024

I decided to learn to code for fun in mid January of this year.


I started by finding a super-duper beginner friendly project here and used that as a guide to make my first practice page on January 18th-ish. I made three more practice pages that day. The next day I stumbled across Frontend Mentor and, consequently, GitHub. I thought they were probably scams set up to steal my identity and download viruses onto my computer. They weren't, so I set myself a new goal of completing the Frontend Mentor QR code component challenge.


I worked toward that for, like, half a week, and then I met this guy named Peter and lost focus for a bit. It turns out Peter is great, but we've been together for six months now and ADHD brain is circling back around to coding. I picked back up with Practice Page Nine yesterday, but I feel like those couple of weeks I spent in January are probably enough HTML to start with, so I moved on to learning CSS.
